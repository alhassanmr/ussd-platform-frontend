# USSD Platform — Frontend

React dashboard for the USSD-as-a-Service platform.

## Features
- Tenant registration & login
- USSD app management
- Visual menu builder
- Integration / webhook setup
- Billing & subscription management
- Super-admin portal (`/admin`)

## Setup

```bash
npm install
npm run dev        # http://localhost:3000
```

Set `VITE_API_URL` to point to your backend (default: http://localhost:8080 via proxy).
